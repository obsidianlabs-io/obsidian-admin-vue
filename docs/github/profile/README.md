# Obsidian Labs

Obsidian Labs builds structured, scalable software foundations for serious products.

Obsidian Labs 致力于为真正要长期演进的产品构建清晰、稳固、可扩展的软件基础。

## Who We Are

We focus on systems that need more than visual polish or short-term scaffolding. Our work emphasizes clear domain boundaries, strong operational safety, maintainable architecture, and production-grade engineering discipline.

我们关注的不是短期可交付的表面工程，而是那些需要清晰边界、稳定运行、长期维护与持续演进能力的系统。

## What We Build

- Enterprise admin platforms
- SaaS application foundations
- Internal operations systems
- Strictly-typed API backends
- Clean frontend and backend integration baselines

- 企业级管理后台平台
- SaaS 应用基础架构
- 内部运营系统
- 严格类型化的 API 后端
- 清晰可维护的前后端集成基线

## What We Value

- Long-term maintainability over short-term convenience
- Explicit architecture over hidden coupling
- Operational clarity over accidental complexity
- Scalable multi-tenant foundations for real business systems
- Strong engineering standards across CI, testing, security, and release workflows

- 长期可维护性高于短期便利
- 显式架构优于隐式耦合
- 运行清晰度优于意外复杂度
- 面向真实业务系统的可扩展多租户基础
- 覆盖 CI、测试、安全与发布流程的工程纪律

## Vision

Obsidian was shaped around a shared direction: leadership, boundary-pushing thinking, depth, precision, and long-term value. We believe good software foundations should be deliberate, resilient, and clear under growth.

Obsidian 的形成源于一条共同的方向: 领导力、突破边界的思考、深度、精准，以及长期价值。我们相信，优秀的软件基础应当是有秩序的、具备韧性的，并且在系统成长后依然清晰可控。

## Build With Intention

We build systems that are:

- clear to extend
- safe to operate
- disciplined under scale
- designed for long-term products, not temporary demos

我们希望构建的系统应当具备:

- 易于扩展
- 运行安全
- 在规模增长下仍然保持纪律性
- 面向长期产品，而不是短期演示
