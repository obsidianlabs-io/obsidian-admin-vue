# Obsidian Admin Vue v1.1.0

A tenant-aware frontend release that strengthens contract-driven development, organization and team management, and production-ready quality gates for the Obsidian admin platform.

## Highlights

- Added tenant-scoped `Organization` and `Team` management pages, search flows, operation drawers, route registrations, and API typings
- Added organization and team fields in user create and edit flows, including team-to-organization constraint handling
- Strengthened frontend and backend API contract validation through generated typings, snapshot checks, and compatibility gates
- Improved user-center maintainability by extracting profile, 2FA, and session logic into dedicated composables
- Cleaned generated API typing workflow to keep CI contract checks deterministic

## Internal Improvements

- Refined role option and assignable role handling for tenant-aware user operations
- Updated OpenAPI and backend DTO typing generation flow for stricter frontend type safety
- Reduced page-level responsibility in complex modules by moving behavior into reusable composables
- Tightened lint, type, and generated-file consistency around contract-driven API integration

## Quality Gates

- ESLint
- Vue TypeScript checks
- API contract snapshot gate
- Frontend and backend contract compatibility gate
- Unit tests
- Conditional Playwright E2E on PR-relevant UI changes

## Verification

```bash
pnpm lint:check
pnpm typecheck:api
pnpm test:unit
pnpm build
```

## Release Positioning

This release establishes a stronger baseline for multi-tenant Vue admin systems that rely on generated API contracts, strict CI gating, and clean integration with `Obsidian Admin Laravel`.
