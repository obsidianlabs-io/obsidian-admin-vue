# Obsidian Admin Vue v1.2.1

A focused frontend patch release that aligns the published Vue client with the Laravel 13 backend lane, refreshes the auth contract, and hardens request/runtime behavior for production use.

## Highlights

- Aligned the generated auth client and frontend auth facade with the backend kebab-case auth routes for token refresh and user info flows
- Added a shared Naive UI provider access layer for programmatic UX surfaces outside component setup
- Hardened request-layer error normalization and JSON response detection against the current Laravel backend responses
- Refreshed the stable release pairing so frontend `v1.2.1` is the coordinated companion for backend `v1.3.1`

## Verification

```bash
pnpm check:ci
pnpm release:pairing v1.2.1
```

## Release Pair

- Frontend: `v1.2.1`
- Compatible backend release: `v1.3.1`
