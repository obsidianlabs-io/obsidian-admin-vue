# Obsidian Admin Vue v1.2.0

A feature release that turns the frontend into a stronger public-facing admin template with generated SDK discipline, preview/runtime hardening, stricter quality gates, and release-grade artifacts.

## Highlights

- Migrated core and system API facades to the generated SDK adapter flow
- Added public docs, public preview, full-stack pairing smoke, visual baselines, accessibility baselines, and Lighthouse preview checks
- Added downloadable release artifacts for production, demo preview, and Pages bundles
- Added stronger release pairing validation, support policy, and launch/demo documentation
- Tightened frontend quality gates to zero-warning linting and stricter CI expectations

## Verification

```bash
pnpm check:ci
pnpm test:preview
pnpm test:fullstack
pnpm test:lighthouse
```
