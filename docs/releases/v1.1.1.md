# Obsidian Admin Vue v1.1.1

A patch release focused on release readiness, contract synchronization, and open-source repository polish for the Obsidian admin frontend.

## Highlights

- Synced generated frontend backend typings with the latest `Obsidian Admin Laravel` DTO boundaries
- Removed persistent generated-typing lint noise by excluding generated API typing artifacts from ESLint checks
- Refined the Chinese and English README homepage structure for better open-source presentation
- Improved repository metadata, release documentation, and GitHub setup guidance for public maintenance

## Internal Improvements

- Updated package metadata to better reflect the current project scope and searchability
- Added GitHub repository metadata and setup checklist documents aligned with the current CI and release workflow
- Kept generated API contract files and release workflow expectations aligned with the backend repository state
