# Obsidian Admin Vue v1.2.2

A frontend maintenance patch focused on architecture cleanup, clearer demo/runtime boundaries, and steadier CRUD drawer behavior.

## Highlights

- Moved the built-in demo backend out of `src/` and guarded demo imports behind runtime checks so production builds stay free of mock-backend code paths
- Extracted auth store side effects into a bootstrap layer to reduce store coupling around route, tab, and websocket lifecycle work
- Standardized CRUD operate drawer form behavior through a shared composable across user, role, permission, tenant, organization, team, and language modules
- Removed the unused `packages/alova` workspace package and fixed package-level ESLint shadowing issues in request factory code
- Stabilized operate drawer async submit tests by waiting for pending promise work before assertions
